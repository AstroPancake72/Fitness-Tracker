# Fitness-Tracker
Repository for a fitness tracker app

Steps for building the project:
1. Clone the repository
2. Enter the project directory
3. Run npm install
4. Enter the backend directory
5. Copy the .env.example file into .env
6. Change the SESSION_SECRET environment variable in .env to a random string
7. In backend: run "node server.js"
8. In frontend: run "npm run dev"
9. In a browser, go to localhost:5173
